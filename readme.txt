A blog theme called Exploore.

== Description ==

Are you a blogger, looking for beautiful free WordPress theme? Exploore should be your ready to go theme for blogging. Perfectly suitable for personal, affiliate, fashion, photography blogging.

* Mobile-first, Responsive Layout with real device
* Post Slider
* Custom Colors
* Custom Header
* Menu Description
* Google Fonts
* WordPress Customizer
* Sticky Post Slider
* Search Button
* The GPL v2.0 or later license. :) Use it to make something cool.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
5. In your admin panel, go to Plugins > and click the Add New button for install " Contact Form 7 ".
6. Go to https://www.themexpert.com/docs/wordpress-themes/exploore for a guide on how to customize this theme.
